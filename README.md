# Senior Connect App Design

Senior Connect is a mobile application designed to support elderly users through simple daily check-ins, emergency SOS alerts, caregiver monitoring, and wellness tracking. The app focuses on creating a safer and more connected environment for seniors living independently.

---

# Project Setup Guide

## 1. Clone the Repository
```bash
git clone https://github.com/liting1123/SeniorConnect.git
```

## 2. Navigate to Project Folder
```bash
cd SeniorConnect
```

## 3. Install Dependencies
```bash
npm install
```

## 4. Start Development Server
```bash
npm run dev
```
